## Usage

> Clone this repo. Run `npm i`, then `npm start`. Go into the client directory, run `npm i`, then run `npm start`.

## Requirements

- Node 0.10.x
- PostgreSQL 9.1.x
- Express 4.17.1.x
- PG 8.1.0.x
- Knex 0.21.x

## Development

### Installing Dependencies

From within the root directory:

```sh
npm i
npm start
```

From within the client directory:
```sh
npm i
npm start
```
